## [1.0.0] - 2026-03-XX

### Added
- Initial release of Odoo AI Assistant
- AI Chat Interface with floating widget
- Smart Field Autocomplete
- Document Intelligence (OCR, data extraction)
- Email Composition Assistant
- Multiple AI Provider Support:
  - OpenAI (GPT-4o, GPT-4o-mini)
  - Ollama (Local AI)
  - Anthropic Claude
- Pre-built prompt templates
- Conversation history
- Document processing for:
  - Invoices
  - Business cards
  - Contracts
  - Receipts
- Settings configuration panel
- Role-based access control
- Dark mode support
- Responsive design

### Technical
- Support for Odoo 16, 17, 18, 19
- LGPL-3 License
- Full i18n support (en, fr, es, de, pt)
